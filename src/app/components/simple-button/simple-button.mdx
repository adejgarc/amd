# Botón
El botón es el elemento de interacción básico. Se utiliza para enviar formularios, confirmar acciones o cerrar diálogos.

## Cuándo usarlo
- **Primary:** Cuando quieres ver mucho azul.
- **Secondary:** Cancelar o cosas del estilo.
- **Danger:** Muy rojo.
- **Clear:** Vacío;
